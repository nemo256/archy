### Using curl (curl is already installed):
```shell
$ curl -O https://raw.githubusercontent.com/nemo256/archy/master/archy
$ chmod +x archy
$ ./archy
```

### Using git:
```shell
$ pacman -Sy git
$ git clone https://github.com/nemo256/archy
$ cd archy
$ ./archy
```
